# 🌱 Fertilizer Prediction System

A Machine Learning-based Fertilizer Prediction System built using **Python, Scikit-learn, and Streamlit**. This application recommends the most suitable fertilizer based on soil properties, crop type, and environmental conditions.

---

## 📖 Project Overview

Choosing the correct fertilizer is essential for healthy crop growth and improved agricultural productivity. This project uses a trained Machine Learning model to predict the most appropriate fertilizer from user-provided agricultural data.

The application provides a simple and interactive Streamlit interface where users can enter soil and crop information and receive an instant fertilizer recommendation.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/ahmadhaseb/Fertilizer-Prediction-System.git
```

### Move to Project Folder

```bash
cd Fertilizer-Prediction-System
```

### Install Required Libraries

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```

---

## 📊 Application Workflow

1. Launch the Streamlit application.
2. Enter soil and environmental parameters.
3. Click the **Predict** button.
4. The trained Machine Learning model analyzes the input.
5. The system recommends the most suitable fertilizer.
